# LockWitness

Owner-controlled Android failed-unlock evidence recorder.

## Purpose

LockWitness is intended to capture and preserve configurable evidence when someone attempts to access the owner's Android device without authorization.

## Planned Features

- Failed unlock monitoring
- Front-camera photo capture
- Optional short video clips
- Optional GPS/location snapshot
- Local evidence timeline
- Optional email/share alerts
- Manual export ZIP/PDF/CSV
- SHA-256 evidence hashing
